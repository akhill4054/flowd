# flowd

Open-source WhatsApp automation infrastructure built on Meta Cloud APIs.

## Project Structure

```text id="1jttys"
flowd/
├── backend/                 # Golang backend services
├── app/                     # Flutter dashboard (web/mobile)
└── docs/                    # Documentation
```

## Backend

* Golang
* Meta Cloud API integration
* Webhook handling
* Workflow orchestration
* Messaging infrastructure

## Dashboard

* Flutter
* Web + mobile support
* Admin and client interfaces
